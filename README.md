# Luminary — Landing Page

A modern, production-grade SaaS landing page built with React + Vite. Designed as a portfolio piece showcasing full-stack development skills.

## ✨ Features

- Dark editorial design with electric lime accent
- Animated hero with interactive dashboard mockup
- Scroll-triggered fade-in animations
- Animated stat counters
- Interactive feature cards
- Responsive layout (mobile-friendly)
- Zero external CSS dependencies — pure custom CSS
- Google Fonts (Syne + DM Mono)

## 🚀 Getting Started

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
npm run preview
```

## 📁 Stack

- React 18
- Vite 5
- Pure CSS (no Tailwind, no UI library)
- IntersectionObserver for scroll animations
- No external dependencies beyond React
